# E-Health Care Management System

A Java Swing application for managing hospital operations, including Doctor approvals, Patient appointments, and Medical records.

## Features
- **Admin Panel:** Approve doctors, view dashboard stats, manage knowledge base.
- **Doctor Portal:** View appointments, diagnose patients, reject appointments.
- **Patient Portal:** Book appointments, view medical history.
- **Security:** Encrypted login, strict input validation.

## Prerequisites
1. Java Development Kit (JDK 8 or later)
2. MySQL Server (XAMPP or WAMP)
3. MySQL Connector JAR

## Setup Guide
1. Import the `database.sql` file into your MySQL/phpMyAdmin.
2. Update the `DB_USER` and `DB_PASS` in the Java files if your database credentials differ.
3. Add the `mysql-connector` JAR to your project libraries.
4. Run `FrontFrame.java` to start the application.
